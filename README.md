# MEAN - Employees List

Simple CRUD in **Angular** and **Typescript**

# Dependencies

- Angular Cli
- Typescript
- MongoDB
- Express
- Babel
- NodeJS
- Bootstrap

## How to install?

1) /server `npm install`
2) /frontend `npm install`

## To run

1) Server: `npm start` on port 3000
2) Front: `ng serve` on port 4200
